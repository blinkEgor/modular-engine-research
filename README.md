# modular-engine-research
## Исследование модульной архитектуры (C++).
---
Этот репозиторий документирует моё исследование по созданию системы сборки движков из независимых бинарных модулей. Включает две итерации подхода, анализ проблем и выводы.
- **Цель:** Изучить возможности динамической загрузки модулей, бинарных интерфейсов (ABI) и создания безопасной модульной экосистемы.
- **Итерация 1:** Два взаимодействующих модуля (BLINK_CORE / BLINK_API), доказавшие принцип работы независимого рантайма.
- **Выводы и следующий шаг:** Анализ ограничений первой архитектуры привёл к проектированию новой системы, основанной на компонентном подходе.
